# Line

Renders a line.

# Usage

```example
<script>
  import { Game, Scene, Line } from 'svelte-phaser'
</script>

<Game width={400} height={400}>
  <Scene key="main">
    <Line
      x={100}
      y={100}
      x1={0}
      y1={0}
      x2={100}
      y2={100}
      strokeColor={0xff0000} />
  </Scene>

</Game>
```
